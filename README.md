# Face Authentication

### Tech Stacks
1. Frontend
- React 18.x
- Material UI
- CSS
- ES6/JSX
- Axios
- React Webcam
- Face-api.js

2. Backend
- Node.js
- Express.js
- Tyepscript
- MongoDB


### How to run
1. ```npm run install```

2. ```npm run server:start```

3. ```npm run client:start```


4. Go to http://localhost:3000
